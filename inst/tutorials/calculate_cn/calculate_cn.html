基本运算

R 里面的运算符号与数学中的基本一致，比如 +、-、*、/。

# 基础四则运算
1 + 2
## [1] 3
3 - 4
## [1] -1
5 * 6
## [1] 30
7 / 8
## [1] 0.875
# 复合运算
(2 + 4) / (5 - 3)
## [1] 3

空格的使用

数字与运算符之间加入空格是出于提高代码可读性的考虑——R 本身有没有空格并不影响运算。下面的代码即使空格不规则，也能正确运行：

(2+4) /  (  5-   3        )
## [1] 3

对于一些旧代码以及来自别人的代码，使用者可以通过 styler、air 等软件包对它们进行一键式统一格式。

除了四则运算外，R 还能完成更复杂的算术：幂运算（^）、取余（%%）、对数（log()、log10()）、指数（exp()）、平方根（sqrt()）、绝对值（abs()）。

10^2
## [1] 100
5 %% 3      # 余数
## [1] 2
log(11)
## [1] 2.397895
log10(100)
## [1] 2
exp(12)
## [1] 162754.8
# 开方两种方式
9^(1/2)
## [1] 3
sqrt(9)
## [1] 3
# 取绝对值
abs(-9)
## [1] 9

三角函数：

sin(pi)
## [1] 1.224647e-16
cos(pi)
## [1] -1
tan(pi / 2)
## [1] 1.633124e+16
acos(1)
## [1] 0
asin(1)
## [1] 1.570796
atan(0.5)
## [1] 0.4636476

R 中预存了一系列常用常量。pi 是其一，还有 letters、LETTERS、month.name、month.abb 等。在下方试一试：

letters
month.name

科学计数法

上例中 tan(pi / 2) 输出 1.633124e+16，这是科学计数法：实数 a（1 ≤ |a| < 10）与 10 的整数次幂之积。字母 e（exponential）连接系数与指数。所以 200 = 2×10² 写作 2e2，0.002 = 2×10⁻³ 写作 2e-3。

布尔代数

R 中布尔代数的两种结果用 TRUE / FALSE 表示（也可简写为 T / F）。在数值环境中，FALSE 视为 0，TRUE 视为 1。三种布尔连接符：&（与）、|（或）、!（非）。布尔”空集”是 NULL。

布尔代数常与不等式搭配：==、!=、>、<、>=、<=。

# 等式
1 == 2
## [1] FALSE
1 == TRUE   # TRUE、FALSE 是赋予逻辑属性的 1 和 0
## [1] TRUE
0 != F
## [1] FALSE
# 不等式
1 != 2
## [1] TRUE
1 > 2
## [1] FALSE
1 < 2
## [1] TRUE
1 >= 2
## [1] FALSE
1 <= 2
## [1] TRUE
# 布尔运算
3 > 2 & 2 > 1
## [1] TRUE
12 & 2 > 2
## [1] FALSE
-3 | 2 > 2
## [1] TRUE

3 > 2 & 2 > 1 先分别判断不等式，再做布尔运算。当只让一个特定值参与布尔代数运算时（如 12 & 2 > 2），R 会进行类似自相等判断。

等于的使用

R 中单个等号 = 留作了赋值符（详见第 4 章），所以表达相等需要用 ==。这种写法也与 !=、>=、<= 保持一致。初学时常会忘记用 ==，多加练习就好。

向量

R 中 c()（concatenate，组合）用来创建向量。对于连续整数，冒号 : 是简写。seq() 是第三种创建方式。

# 方式一：c()
a  <- c(11, 12, 13, 14, 15)

# 方式二：冒号（仅适用于连续整数）
a1 <- 11:15

# 方式三：seq()
a2 <- seq(from = 11, to = 15, by = 1)
a3 <- seq(11, 15)

# 验证三种方式生成的结果是否完全一致
a == a1
## [1] TRUE TRUE TRUE TRUE TRUE
a == a2
## [1] TRUE TRUE TRUE TRUE TRUE
a == a3
## [1] TRUE TRUE TRUE TRUE TRUE

索引

方括号 [] 用来从向量中提取元素。

a <- c(11, 12, 13, 14, 15)

# 单个元素
a[2]
## [1] 12
a[5]
## [1] 15
# 多个元素
a[c(3, 4)]
## [1] 13 14

赋值符 <-

<- 是 R 中最常用的赋值符。箭头指向左边，表示把右边的对象”存进”左边的名字。

R 几乎是目前流行编程语言中唯一使用 <- 作赋值符的，其他语言多用单个等号 =。这要追溯到 R 的前身 S 语言。上世纪六十年代，名为 APL（A Programming Language）的编程语言流行起来，APL 含有很多特殊字符，配套键盘右上角有一个左箭头键，<- 由此而来。APL 没落后，这个习惯被保留进了 S，进而保留进了 R。

R 为该符号设定了快捷键：Windows / Linux 上是 Alt + -，Mac 上是 Option + -。

向量四则运算

加减法：对应元素相加减：

\[\vec{a} \pm \vec{b} = (a_1 \pm b_1,\; a_2 \pm b_2,\; \ldots,\; a_n \pm b_n).\]

a <- c(11, 12, 13, 14, 15)

# 向量与标量加减
b <- a + 4
c <- a - 0.5

# 向量加减
a + b
## [1] 26 28 30 32 34
a - c
## [1] 0.5 0.5 0.5 0.5 0.5

乘法上，* 是元素积（哈达玛积，Hadamard product），%*% 是内积（点乘），%o%（或 outer()）是张量积（外积）。

a <- c(11, 12, 13, 14, 15)
b <- a + 4

# 元素积（Hadamard product）
a * b
## [1] 165 192 221 252 285
# 内积（点乘）
a %*% b
##      [,1]
## [1,] 1115
# 外积/张量积
a %o% b
##      [,1] [,2] [,3] [,4] [,5]
## [1,]  165  176  187  198  209
## [2,]  180  192  204  216  228
## [3,]  195  208  221  234  247
## [4,]  210  224  238  252  266
## [5,]  225  240  255  270  285

向量的”非常规”运算

数学上向量没有”除法”运算。但在 R 中，a / b 具有明确定义，代表元素逐项相除。

a <- 11:15
b <- a + 4

a / b
## [1] 0.7333333 0.7500000 0.7647059 0.7777778 0.7894737

当两个向量长度不一致时，R 会自动启动回收原则（recycling rule）：短向量被循环使用，直到匹配上长向量的长度。比如 e <- c(1, 2) 配 a 长度为 5 时，e 被扩成 (1, 2, 1, 2, 1)。

a <- 11:15
e <- c(1, 2)

a * e
## [1] 11 24 13 28 15

R 会同时给一个警告。

c() 也可以创建字符型向量——记得每个元素加双引号：

aSentence <- c("I", "am", "from", "People's Republic of China", ".")
aSentence
## [1] "I"                          "am"                        
## [3] "from"                       "People's Republic of China"
## [5] "."

现学现练

练习一：根据下面的赋值，请解释并推算每个表达式的结果。

a <- 11:15
e <- c(1, 2)

a - e
a / e
# e 被回收为 (1, 2, 1, 2, 1) 以匹配 a 的长度。
#
# a - e: 11-1, 12-2, 13-1, 14-2, 15-1 = 10 10 12 12 14
# a / e: 11/1, 12/2, 13/1, 14/2, 15/1 = 11 6 13 7 15

练习二：推演干支纪年系统

干支纪年由十天干和十二地支组合而成。理论上 120 种组合，实际只取 60 种构成完整甲子：阳干（奇数位）只配阳支，阴干（偶数位）只配阴支。

  • 十天干：甲、乙、丙、丁、戊、己、庚、辛、壬、癸
  • 十二地支：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

提示

  • seq() 配合 %% 可以分阴阳。
  • outer() 生成两两组合。
  • paste0() 可以把字符串粘起来，比如 paste0("甲", "子") 返回 "甲子"。
tiangan <- c("甲", "乙", "丙", "丁", "戊", "己", "庚", "辛", "壬", "癸")
dizhi   <- c("子", "丑", "寅", "卯", "辰", "巳", "午", "未", "申", "酉", "戌", "亥")
tiangan <- c("甲", "乙", "丙", "丁", "戊", "己", "庚", "辛", "壬", "癸")
dizhi   <- c("子", "丑", "寅", "卯", "辰", "巳", "午", "未", "申", "酉", "戌", "亥")

tg_yang <- tiangan[seq(tiangan) %% 2 == 1]
tg_yin  <- tiangan[seq(tiangan) %% 2 == 0]

dz_yang <- dizhi[seq(dizhi) %% 2 == 1]
dz_yin  <- dizhi[seq(dizhi) %% 2 == 0]

outer(tg_yang, dz_yang, FUN = paste0)
outer(tg_yin,  dz_yin,  FUN = paste0)

矩阵

向量是一维的，矩阵是两个以上向量组成的二维世界。

\[ A_{2 \times 2} = \begin{pmatrix} 1 & 3 \\ 2 & 4 \end{pmatrix}. \]

# 创建一个 2 行的矩阵，数据按列填充
matrix(c(1, 2, 3, 4), nrow = 2)
##      [,1] [,2]
## [1,]    1    3
## [2,]    2    4

nrow 是 “number of rows”；ncol 是 “number of columns”。两者只指定一个即可，另一个 R 会自动算。R 按”列优先”（column-major order）填充矩阵。回收原则同样适用：

matrix(1:3, nrow = 2)
##      [,1] [,2]
## [1,]    1    3
## [2,]    2    1

矩阵元素通过行 × 列索引。位置留空表示”全部”。

A <- matrix(1:3, nrow = 2)

A[1, 2]
## [1] 3
A[1, ]
## [1] 1 3
A[, 2]
## [1] 3 1

单一矩阵运算

转置（transposition）将行列互换。在 R 中是 t()。

A <- matrix(1:3, nrow = 2)

A
##      [,1] [,2]
## [1,]    1    3
## [2,]    2    1
t(A)
##      [,1] [,2]
## [1,]    1    2
## [2,]    3    1
t(t(A))
##      [,1] [,2]
## [1,]    1    3
## [2,]    2    1

嵌套调用的逻辑

t(t(A)) 是由内而外计算：R 先算 t(A)，再把结果作为外层 t() 的输入。这与数学上的复合函数 \(g(f(x))\) 原理一致。

C <- matrix(6:1, nrow = 2)

解方程组

很多统计方法本质都是解线性方程组。考虑：

\[ \begin{cases} a + b + c = 6, \\ 3a + 2b + 4c = 9, \\ 2a + 2b - 6c = 4. \end{cases} \]

矩阵 B 为系数，\(\vec{y}\) 为右侧，\(\vec{x}\) 是解：\(\vec{y} = \mathbf{B}\vec{x}\)。这与回归方程 \(y = \beta_0 + \beta_1 x_1 + \cdots + \beta_n x_n + \epsilon\) 形式相似。R 中 solve() 直接求解：

B <- matrix(c(1, 3, 2, 1, 2, 2, 1, 4, -6), nrow = 3)
B
##      [,1] [,2] [,3]
## [1,]    1    1    1
## [2,]    3    2    4
## [3,]    2    2   -6
y <- c(6, 9, 4)

x <- solve(B, y)
x
## [1] -5 10  1
