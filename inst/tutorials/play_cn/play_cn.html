面向对象编程

面向对象编程（OOP）通过类（class）和对象（object）来组织和封装数据。C++、Java、Python 等都采用 OOP，R 也是如此。

  • 类：对象的结构，包含其属性和方法。
  • 对象：类的具体实例，如”中国”之于”国家”，“R”之于”字母”。
  • 方法：类内定义的命令。
  • 属性：描述对象的特征，比如人的发色。

OOP 语言通常还共享封装、多态、继承、安全这四个特性。

多态（Polymorphism）让一个命令可以对不同对象进行不同处理。最典型的例子是 summary()：

summary(wvs7)
summary(wvs7$age)

函数

函数可以理解为状语后置的祈使句：

<函数名>(<目标数据>, <条件 1>, <条件 2>, ...)

OOP 之信仰：不触碰数据，让命令去做。

light <- function(finger){
  shadow <- finger + 5
  shadow
}
handShadow <- light(finger = 3)
handShadow

赋值符 <-

<- 是 R 中最常用的赋值符，等同于 assign() 命令的默认行为：

<名称> <- <对象>

aValidObject <- 1:5
aValidObject
## [1] 1 2 3 4 5

R 中赋值有四种符号：assign()、<-、<<-、=。<- 最常用——指向直观，且不和 =、== 混淆。快捷键：Windows / Linux 上是 Alt + -，Mac 上是 Option + -。

a <- 12
25 -> b   # 右箭头也可以

命名规则

对象名需满足：

  1. 不以数字开头（错误：1stday）。
  2. 除 . 和 _ 外没有特殊符号（错误：M&M）。
  3. 区分大小写（X != x）。
  4. 不要重写内置命令（避免 list <- c(1:5)）。
  5. 表意命名。

记录（数据存储）

向量

c() 用来创建向量。一个向量内的元素必须共享同一种数据类型。

数字向量

vec_integer <- c(1, -2, NA)
vec_double  <- c(1.5, -2.34, 1/3)

NA 表示 not available——R 中的缺失值。

字符向量

vec_chr <- c("R", "is", "useful.")
vec_chr
## [1] "R"       "is"      "useful."

类别向量（factor）

类别向量存储分类变量，包含一个固定的”水平”集合（levels）。

vec_fac  <- factor(c(1, 2, 2, 3))
vec_ord  <- ordered(c(1, 2, 2, 3))
vec_fac2 <- factor(c(1, 2, 2, 3),
                   levels = c(3, 2, 1),
                   labels = c("Apple", "Pear", "Orange"))

vec_fac
## [1] 1 2 2 3
## Levels: 1 2 3
vec_ord
## [1] 1 2 2 3
## Levels: 1 < 2 < 3
vec_fac2
## [1] Orange Pear   Pear   Apple 
## Levels: Apple Pear Orange

时间向量

R 中时间有两种表示：

  • as.POSIXct()——数值存储，记录从 1970-01-01 到当前时间的秒数。
  • as.POSIXlt()——列表存储，时间的每一部分（年、月、日等）都是列表元素。
ct <- as.POSIXct("2026-04-13 10:11:12")
lt <- as.POSIXlt("2026-04-13 10:11:12")

unlist(ct)
## [1] "2026-04-13 10:11:12 CST"
unlist(lt)
##    sec    min   hour   mday    mon   year   wday   yday  isdst   zone gmtoff 
##   "12"   "11"   "10"   "13"    "3"  "126"    "1"  "102"    "0"  "CST"     NA

时间数据的瑞士军刀：lubridate

ymd("20221016")
## [1] "2022-10-16"
mdy("10-16-2022")
## [1] "2022-10-16"
dmy("16/10/2022")
## [1] "2022-10-16"
ymd_hms("2022-10-16 09:00:00", tz = "Etc/GMT+8")
## [1] "2022-10-16 09:00:00 -08"

数据框（data frame）是特殊的列表（或矩阵），每列是一个变量（长度相等），每行是一个观测：
